# Project Description

The Smart Agriculture AI System is an intelligent web-based application developed to assist farmers in making better agricultural decisions using Artificial Intelligence. The system combines Machine Learning and Deep Learning techniques to solve multiple agriculture-related problems through a single platform.

The application provides recommendations and predictions based on user input, helping farmers increase crop productivity, reduce fertilizer misuse, detect plant diseases at an early stage, and estimate crop yield accurately.

The project has been developed using Python, Streamlit, Machine Learning, Deep Learning, TensorFlow, Scikit-learn, and various data analysis libraries.

---

# Project Objectives

* Detect plant diseases using Deep Learning.
* Recommend suitable fertilizer for crops.
* Predict crop yield using Machine Learning.
* Provide an easy-to-use web interface.
* Help farmers make data-driven agricultural decisions.

---

# Modules of the Project

## 1. Plant Disease Detection

This module uses a Deep Learning model based on VGG16 Transfer Learning to classify diseases from plant leaf images.

### Features

* Upload leaf image
* Automatic disease prediction
* Healthy leaf detection
* High prediction accuracy
* Disease confidence score

### Technologies Used

* TensorFlow
* Keras
* VGG16
* Transfer Learning
* Fine Tuning
* OpenCV
* Pillow

---

## 2. Fertilizer Recommendation

This module recommends the most suitable fertilizer according to crop type, soil type, soil nutrients, temperature, humidity, and other environmental conditions.

### Input Parameters

* Crop Type
* Soil Type
* Nitrogen (N)
* Phosphorus (P)
* Potassium (K)
* Temperature
* Humidity
* Moisture

### Output

Recommended Fertilizer

---

## 3. Crop Yield Prediction

This module predicts the estimated crop yield using Machine Learning Regression techniques.

### Input Parameters

* Crop Type
* Soil Type
* pH
* Temperature
* Humidity
* Wind Speed
* N
* P
* K
* Soil Quality

### Output

Estimated Crop Yield (Tons/Hectare)
